## ECG繪圖
 * 國際上公認的可作為標準的心電資料庫有三個，分別是美國麻省理工學院提供的MIT-BIH 資料庫，美國心臟學會的AHA資料庫以及歐洲ST-T心電資料庫。
 * MIT-BIH為研究心律失常的資料庫。
 *  MIT-BIH 為了節省檔長度和存儲空間，使用了自定義的格式。
 * 此程式轉換的訊號格式為format212
## 心電圖紀錄的檔案類型 
  * 頭檔[.hea]-資料格式與病患基本資料
  * 數據檔[.dat]
  * 註釋檔[.atr]-心電專家對相應的心電信號的診斷資訊
